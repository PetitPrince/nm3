Newton's Manuscript Mystic Manufactorum

What is that ?

Encode any text using a simple substitution method. By default it uses Newton alchemical symbol. When using the output in for your own usage outside of this page, please install the Newton Sans font so that the Zodiacal sign are not transformed into ugly emoji (♋ instead of ♋)

Please note that mutiple word substitution is possible, and that the order have no importance. Case is ignored.

Example:

With this mapping:

{
    "a": "♁",
    "b": "♆",
    "c": "♋",
    "d": "ʒ",
    "wizard": "🧙‍♂️",
    "wizard who shall not be named": "🧛‍♀️"
}

An this input:

I am a wizard. I don't like the wizard who shall not be named.

The output will be:

i ♁m ♁ 🧙‍♂️. i ʒon't like 🧛‍♀️. he's very me♁n.
